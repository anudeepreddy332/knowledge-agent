
# Example knowledge base entry

## RAG (Retrieval-Augmented Generation)
RAG combines a retrieval step with a generative LLM. The retriever fetches relevant chunks from a knowledge base, and the LLM uses them as context to answer.

## Hybrid search
Hybrid search merges BM25 (keyword) and dense vector (semantic) retrieval. Reciprocal Rank Fusion (RRF) is used to combine rankings.
